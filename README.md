## Useage
1. clone this porject.
```bash
git clone git@github.com:admin1949/remake.git
cd remake
```

2. installation dependence.
```bash
yarn install
```
Or
```bash
npm i
```

3. Start Local server
```bash
yarn dev
```
Or
```bash
npm run dev
```

local Server: [http://localhost:3000](http://localhost:3000)
